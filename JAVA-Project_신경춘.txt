		1.관리자 로그인 :1)신규 학생 추가		신규등록시 로그인 아이디 부여
				2)신규 교수 추가
				3)학생 정보 삭제 - remove
				4)교수 정보 삭제
				5)모든 교수 정보 보기
				6)모든 학생 정보 보기
				7)교수 정보 찾기-hashmap
				8)학생 정보 찾기-hashmap
				
		2.교수 로그인 : 로그인 한 교수 정보 출력
				1)담당 과목 등록
				2)담당 과목 학생 정보 보기
				3)정보 수정

		3.학생 로그인 : 로그인 한 학생 정보 출력
				1)수강 과목 신청
				2)정보 수정
							학생 기본 정보는 파일 입력,
							수강 과목 입력은 키보드 입력
